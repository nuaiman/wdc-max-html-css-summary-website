HTML & CSS Basic Summary

HTML Summary

HTML (Hyper Text Markup language) is used to define our page content, structure and meaning. You don't use it for styling purposes. Use CSS for that instead!

  • HTML uses "elements" to describe (anotate) content
  • HTML elements typically have an opening tag, content and then a closing tag
  • You can also have void (empty) elements like images
  • You can also configure elements with atributes
  • There is a long list of available elements which you'll learn over time, no worries.

Learn more about all available HTML elements on MDN HTML elements reference

CSS Summary

CSS (Cascading Style Sheets) is used for styling your content.

  1. Styles are assigned by property-value pairs
  2. You can assign styles via the "style" attribute
  3. To avoid code duplication, you typically use global styles (e.g. via the "style" element) instead
  4. Altternatively, you can work with external styllesheet files which you "link" to
  5. When working with CSS, concepts like "inheritence", "specificity" and the "box-model" should be understood.

Learn more about all available CSS properties and values on MDN CSS property reference
